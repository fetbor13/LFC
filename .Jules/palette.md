## 2025-01-20 - Structural ARIA patterns for custom modals
**Learning:** Custom interactive elements like custom modals require explicit structural ARIA roles (e.g. `role="dialog"`, `aria-modal="true"`) to be correctly identified by screen readers, which won't otherwise be able to communicate the element's purpose to the user. It is also important to label inputs where visible text labels are missing, and to add `aria-label` to icon-only buttons (like modal close buttons or theme toggles).
**Action:** When implementing or modifying custom modal structures and icon-only buttons, always explicitly provide `role="dialog"`, `aria-modal="true"`, link the modal to its heading using `aria-labelledby`, and ensure icon buttons and placeholder-only inputs have an explicit `aria-label`.
